# Nihal Ahmed Portfolio Website

A responsive static portfolio website inspired by the section structure of hasifofficial.com: hero greeting, about section, skills, working experience, education, featured work, and contact call-to-action.

## Files

- `index.html` - main portfolio page
- `styles.css` - responsive UI, typography, layout, and visual styling
- `script.js` - mobile menu and dynamic year
- `Nihal-resume.pdf` - downloadable resume

## GitHub Pages hosting

1. Create a new GitHub repository, for example `nihal-portfolio`.
2. Upload all files from this folder to the repository root.
3. Go to **Settings** → **Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Choose `main` branch and `/root`, then save.
6. Your site will be available at `https://YOUR-USERNAME.github.io/nihal-portfolio/`.

For a user/organization site, name the repository `YOUR-USERNAME.github.io` and GitHub Pages will publish it at `https://YOUR-USERNAME.github.io/`.

## Customize

- Replace the `N` portrait placeholder with your photo by adding an image and editing the `.portrait` block in `index.html`.
- Update project descriptions or links inside the `#work` section.
- Change colors in `:root` variables inside `styles.css`.
